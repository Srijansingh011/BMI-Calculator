# 🧮 BMI Calculator

A simple BMI (Body Mass Index) calculator built using Python in Jupyter Notebook.

---

## 📌 Features

* Takes user input (height and weight)
* Calculates BMI
* Displays BMI category:

  * Underweight
  * Normal
  * Overweight
  * Obese

---

## 🚀 How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/bmi-calculator.git
   ```

2. Open the notebook:

   ```bash
   jupyter notebook
   ```

3. Run all cells in `BMI Calculator.ipynb`



## 🧠 Formula Used

BMI = weight (kg) / (height (m))²

---

## 📂 Files in this Project

* `BMI Calculator.ipynb` → Main notebook
* `BMI Calculator.py` → Python script version (optional)
